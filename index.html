The end user will be provided a daily report which showcases the amount of nutrition consumed per day such as, calorie intake, carbs, protein, and fat etc.

We plan to establish this app using appypie.com, which requires no knowledge in programming whatsoever. Since this is a group project, we will all be splitting off the tasks so that everyone gets assigned an equal amount of work which is to be completed through a shared document on Google Docs. We may need to research and analyse some statistics as to what users would prefer to be incorporated in the app. This can be achieved by looking at reviews of similar products on the market as there will be common errors reported that we could prevent from happening. Professional dieticians and chef’s advice may also be an essential human resource as they are able to provide more in-depth information about each recipe.

We plan to release the app on IOS and android platforms as well as a website that includes the same features as the mobile devices. By releasing the app on a variety of commonly used platforms, it helps us to garner a larger number of users in contrast to only releasing it on a single platform. If this project is successful, it will provide many long-term implications such as developing good habits which serves to diminish poor dietary habits.
